# 🔐 Simple Password Generator

## 📌 Project Overview

The **Simple Password Generator** is a Python-based application that generates secure and random passwords. It helps users create strong passwords containing uppercase letters, lowercase letters, numbers, and special symbols to improve online security.

## 🎯 Objective

The main objective of this project is to generate strong and secure passwords based on the user-defined password length and ensure better protection against unauthorized access.

## ✨ Features

✅ Generates random passwords instantly
✅ Includes uppercase and lowercase letters
✅ Uses numbers and special characters
✅ User can choose password length
✅ Displays password strength (Weak/Medium/Strong)
✅ Easy to use and beginner-friendly

## 🛠️ Technologies Used

* 🐍 Python
* VS Code
* Git & GitHub

## 🚀 How to Run the Project

1. Open the project in VS Code.
2. Open Terminal.
3. Run the following command:

```bash
python password_generator.py
```

4. Enter the desired password length.
5. Get your secure password instantly.

## 📷 Sample Output

```text
Enter password length: 12
Generated Password: A@8kL#9mX!2p
Password Strength: Strong
```

* Improves password security
* Saves time in creating passwords
* Generates unique passwords every time
